# Mobile-Sales-Dashboard_PowerBI
Developer a Power BI dashboard using 3,800+ mobile sales records to analyze total sales, quantities sold, customer behavior, and performance across cities, brands, models, and payment methods.

Highlights
- Tracks total sales, total quantity, transactions count, and average order value.
- Shows city-wise sales on an interactive map.
- Breaks down quantity trends month by month.
- Displays customer ratings distribution.
- Compares payment method usage (UPI, Credit Card, Debit Card, Cash).
- Highlights top selling mobile models and brand-wise totals.
- Includes slicers for months, mobile model, brand, payment method, and day name.
- Fully cleaned and structured dataset with 14 columns and 3,800+ rows.
